## Installation

1. Run this command in the command line

  

```bash

$ git clone https://github.com/Luffy909/Topgg-Voter.git

```

2. Now you'r done

  

## How to run ?

  

1. go to command line/terminal and run

  

  

```bash

$ cd Topgg-Voter

```

  

  

2. then you have to install required packges by using this command bellow
  

```bash

$ npm i

```

3. now you need to add tokens to you'r tokens.txt
- go to tokens.txt
- add tokens
  

```bash

$ node index.js

```
